basics of programming

a variable stores data of a certain type
a function executes a series of precise actions
a program is run in a linear fashion
// a double slash indicates a comment
the answer is online
a variable is if we want to keep track of a particular state
a function is if we want to do something in particular. but in order for the function to work, we need to give it parameters, which we put between ( and )
the internet was built by and for programmers, so the canonical resource is always online

basics of digital drawing

setup() and draw() // essential functions of processing
rect(x, y, w, h) / ellipse() / line() // shape functions
fill() and stroke() // color functions
size() and background() // application functions
you always need to put your code between { and }
color is based on RGB values (red, green, blue) and goes from (0, 0, 0) black to (255, 255, 255) white

fundamentals of digital art

- procedure
- repetition
- chance / emergence
- translation
set some rules and find beauty within them
find quality through quantity
remove agency and authorship by allowing random events, and use the audience's ability to imagine and project unto chaos
point at the essence of something by translating it through different media
