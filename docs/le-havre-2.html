types of data

CSV - comma separated values (organized in rows and columns)
JSON - javascript object notation (organized in arrays and variables)
CSV looks like an excel spread sheet, and is useful for fairly simple data
JSON looks more like code, and can be very powerful for complex datasets

web components

three files: html, css and javascript
html is the backbone
css is the skin
javascript is the soul
hypertext markup language
due to our limited time, we will just look at HTML, but most websites need CSS and JS too
CSS is what allows us to decide "this particular tag will be in green" or "this particular tag will be highlighted when the visitor mouses over it"
Javascript is responsible for everything that is interactive (pop-ups, videos, reveal/hide menus, etc.)

focus on html

html works with opening and closing tags:
<p> my content </p>
tags can have attributes:
<p class="theCategory"> my content </p>
html is a "nested-tree structure"
the attribute is anything that goes inside a tag, such a class, href, src
you see these when you inspect the source of an HTML page

scraping

scraping is the act of extracting information from web pages at A LARGE scale
to scrape, we must understand how a particular HTML page is structured
to do that, we do a right-click on the element that we're interested in on the page and select "Inspect Element...", which will show it to us as its HTML form. we can then find its particular tag or class
